# T5 in bert4keras
整理一下在keras中使用T5模型的要点，尤其是中文场景下的使用要点。以多国语言版mT5为例。

博客链接：https://kexue.fm/archives/7867

## 权重下载

首先，要想办法下载Google开放的权重，最简单的方式，是找一台能科学上网的服务器，在上面安装gsutil，然后执行
```shell
gsutil cp -r gs://t5-data/pretrained_models/mt5/small .
```
